# 自学与搜索方法

## 搜索
大学的学习很大程度依靠自学，如何高效率的自学和搜索的技术分不开。

遇到一个Bug不知道怎么解决？

遇到一个知识点不知道如何从一堆网页中直接搜到自己想要的？

如何避免点进去半桶水、误人子弟的教程？

这里可以告诉你。

### 浏览器和搜索引擎
首先，你需要一个**好的浏览器**。

这里建议选择下载**Chrome或者Firefox**，不要使用360搜索之类的，尤其是国产的乱七八糟的浏览器！

ps. 可能大学之前，你用的一直是360浏览器之类的。但是你用一下Chrome或者Firefox体验一下，这两款浏览器可以更好的支持网页插件（你慢慢会知道这个多么有用的），而且UI也很好看，点进去的新标签页也没有那么多广告，清爽很多。总之强烈安利。

如果你一时半会感觉不出来有什么区别，你用多了就会知道了。

其次，如果你是计软网的同学，我建议你在搜索专业有关的知识时，**放弃百度，使用Bing或者google（如果你能翻墙的话，当然，一般bing就够了）**。我这里简略的说一下为什么不要用百度：

1. 百度搜索计算机方面的知识，你的搜索页面会被海量的csdn等垃圾信息污染。当然我不是针对csdn，不得不说用百度搜索计算机方面的知识真的会搜出来很多鱼龙混杂的知识，而辨别这些实在是浪费时间。例如csdn这个网站，上面确实也有一些优质的帖子，但是这上面互相转载、半桶水就乱发帖子的现象太多。

   群里已经有很多被csdn荼毒的同学，他们一般在群里发截图问：“这样为什么运行不了呀”，然后我们一眼就能看出来是csdn上面直接复制粘贴的，他们运行一下发现不行，于是直接发到群里问。为了避免自己也成为这样的人，请远离csdn，就算使用，也一定要搞懂你运行的每一个命令的每一个参数到底是干啥用的。

2. 计算机行业你必须承认国外发展的比国内发达，而且专业的人员都会在外网例如stackoverflow等网站上讨论，当你搜索一些国外开源项目官网时，百度也经常搜不出来，而Bing就可以搜出来

3. 总而言之，对于计软网的同学，当你无法翻墙的时候，Bing是你在国内能用到最好的搜索引擎了

### 搜索习惯
不要搜索“为什么xxxx?”"怎样xxx"。

你需要了解搜索引擎的搜索逻辑（你可以自己去学习一下）。我这里简单的直接说一下搜索方法：**搜索空格隔开的关键词**。

例如你原本想搜索“如何使用wireshark”，那么建议你换成搜索“wireshark”，然后去官网里面找document或者quickstart。

如果你本来准备搜索“如何给Arduino IDE配上esp8266的配置”，那么我建议你搜索“arduino esp8266”。
当然还有更多高级搜索技巧，建议你自己在搜索的过程中学习~

### 翻墙
翻墙有什么用？

可能你会想，我上大学前也不会翻墙，觉得不会翻墙也没感觉有什么影响。

可能你会想，哎呀，墙外面的花花世界感觉有很多不好的信息，不翻墙挺好的。

但是我要告诉你，如果你不翻墙，就像一个人一直在一个山里的小村庄里面。只要当你出去看上一眼，你可能就再也无法忍受只能呆在小村庄里面的生活。

我在会翻墙之前，也没觉得这个有什么。但是会了之后，我学习的速度大幅增加，我觉得翻墙已经成为必备技能了。

上github要翻墙，上stackoverflow等外国技术网站要翻墙，去很多项目官网要翻墙...没有翻墙，你无法想象你错过了什么。

对我而言，我认为翻墙是大学生（尤其是计软网）的必备技能，当然很多人并没有机会学习，因为要想学习翻墙知识，国内是没有的（都被和谐了），只有你翻出去才能知道怎么翻出去。

我不会在这里面直接讲如何翻墙的，建议来协会找学长学姐问（实际上学习很多东西，如果你不懂，我都建议你直接来协会）。
