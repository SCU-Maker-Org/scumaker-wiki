# Linux 与服务器入门

我可以很明确的告诉你们如果你们只照着学校教的课程学习，大学四年学完应该都碰不到服务器，甚至不会连接登录

然而服务器是一件很好玩的东西，例如：

你学完前端有没有想过，拥有一个自己的域名，然后把自己的博客放上去，而不是只能放在本地的静态页面？

你有没有想过摆脱每天手动健康打开，实现自动打卡？

你有没有想过实现翻墙自由，自己搭个梯子？

这些，服务器都可以帮你完成。

- Linux
服务器入门需要少量的Linux知识，建议先看Linux新手教程一个小时大概了解一下即可

- 拥有一台服务器
可以去腾讯云、阿里云买一个轻量应用服务器linux系统香港地区

- 拥有一个自己的域名
可以腾讯云买域名，也可以去freenom自己弄一个免费的域名（有效期一年，一年之后可以续）

- 配置DNS
可以去DNSPOD配置dns解析

- 刷系统、ssh连接、公钥登录
可以看看我的博客[买了一台服务器之后的基操](../../00-wiki/external-links.md)

另外，为什么要刷系统？因为你买来的系统你不知道腾讯\阿里往里面有没有装自己的后台监控，自己刷一遍把这些东西刷掉

- 进阶
当你ssh root@域名 登上自己的服务器后，你对服务器才刚刚入门。服务器有很多玩法，下面列出一些：

1. QQ机器人
github搜索OPQBOT

2. 自己搭博客
可以选择wordpress博客，有很多教程（也有一种不用服务器搭博客的方法，可以搜索github.io搭博客）

3. docker
一份docker配置文件，随处使用，容器式环境。我的wordpress博客就是docker里面搭的

4. nginx
端口转发，负载均衡

5. 搭梯子
不多说

6. 挂脚本24小时跑
例如自动打卡脚本，每天可以定时打卡

7. else
