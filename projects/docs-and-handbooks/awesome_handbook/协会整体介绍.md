# SCU Maker 协会介绍

开源硬件协会是四川大学计算机学院领导下的学术性社团之一，成立于2015年。

“团结，进取，求实，创新”是我们协会的宗旨。在社团未来的发展中，我们将始终致力于创建一个四川大学内的硬件软件结合领域的交流平台，创建一个属于创客人的家，使同学们能够在此获得开源硬件领域内的最新成果和消息。 


## 什么是开源硬件与创客

所谓开源硬件指的是与自由及开放原始码软件相同方式设计的计算机和电子硬件。开源硬件开始考虑对软件以外的领域开源，与创客同属于开源文化的一部分。

 创客作为开源文化的重要参与者它源于英文Maker，指的是不以盈利为目标，努力把各种创意转变为现实的人，是热衷于创意、设计、制造的个人或群体。创客包含了软件开发者、艺术家、设计师等诸多领域的代表，他们均以用户创新为核心理念。他们是坚守创新，持续实践，乐于分享并且追求美好生活的人。 


## 协会建设与项目

基地自建立以来，吸纳了大量来自四川大学各个学院各个专业的软硬件爱好者、具有较强创新能力且乐于学习的同学，吸引了大批团队在基地内开展创新实践，形成了极其浓厚的创新创业氛围。 基地有效借助了指导老师的力量，在他们的指导下先后成立了智能寝室、“极市”无人超市、校园无人车、基于e-paper的多功能显示系统、火灾监控系统等多个创新团队，并制定了多项利用基地设备制作文创产品的计划，将成为四川大学第一个售卖同学自己制作的产品的店铺。

另外，指导老师也热情地定期为基地成员开展相关技术讲座，基地具有较强软硬件技术的同学们也带领新来的同学学习相关知识，使得基地不仅是一家有盈利目的的组织，更是一个传授技术、培养人才的组织。 

在学校的支持下，基地不断更新各种硬件设备，努力为同学们提供最好的创新实践环境，现已拥有大型电子屏、3D打印机、服务器等设备，并根据同学们的建议购进新设备，以同学们的利益为重。现基地已经先后多次组织校内外人士的参观活动，赢得了极高的赞许。 开源硬件协会不仅仅是负责对会员进行学术指导以及开展实践。

## 活动与学习氛围
我们还会在学习之余，开展各种丰富的素拓活动，用生动有趣的方式培养同学们对开源硬件的兴趣，营造一种“玩”的学习环境。“玩”是一种状态：放松、惬意、自由。“玩”是一种实践：不但要动手，还要动脑。“玩”是一种分享：与他人一起玩，才更有趣，更有意义。“玩”是一种境界：当越来越多的人加入创客一起玩时，世界将因“玩”而改变。“玩”式学习还有助于会员们建立深厚的感情，促进协会内部的和谐发展。
