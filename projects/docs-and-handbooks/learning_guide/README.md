# 新人学习指南

## 前言

当你看到这个仓库的时候，**恭喜你成为SCU Maker协会内部的一员！**

SCU Maker的中文名叫开源硬件协会，但是实际上我们很想改掉这个中文名(；′⌒`)因为**我们真的不是只做硬件**

我们协会作为横跨**计软网电子机械自动化艺术历史等**专业的大社团，社团中有**各路大神**。

这里不仅有会写小程序的，会玩服务器的，会自动抢课的，会玩无人机的，会玩船的，会玩机器人的，会翻墙的，会玩AI的...还有会画画的，会剪辑的，会玩VR游戏的。

**在SCU Maker，你甚至可以学开源硬件。**



从前，我大一作为萌新进来的时候，

踏进协会的门口，发现协会的学长做的东西我全都不懂，最入门级别的项目对我来说都是很难，

加了协会的群，发现群里面的大神聊天我也听不懂Orz

作为迷茫的萌新，不要怀疑自己加协会的意义~



从前：

Q：学长学长，我该怎么入门呢？

A：可以先拿个arduino套件自己玩一下

然后拿了套件做了几个示例之后发现还是不会做项目，最后不了了之...

为了改变协会萌新不知道该学什么、怎么学的状况，我开始编写这个仓库，希望对大家有所帮助。

当然，如果看完你还是不知道该学什么，可以**来协会**找我们聊聊~我们就在商业街极市后面

加入SCU Maker你不会失望！
