# 编程语言入门

## 前端语言

- html
- css
- JavaScript

这三个一般是一起学习的，不要感觉三种语言学的很多，实际上静态页面html + css足矣，这两个要入门的话看几个小时的文档教程都已经差不多够用了，学起来很快

JavaScript是给网页加上动态效果的，比如我要在网页上面写个井字棋小游戏，就要用到JavaScript，而如果我只要写一个棋盘，html + css就够了

学习html + css可以通过 菜鸟教程等教程网站学习，建议看文字版的网站教程学习而不是看bilibili，除非你真的看不懂学不下去了，个人觉得看文档比看视频学得快，当然，我入门的时候也是看的视频。。。当时还不知道看文档学的方法

然后你可以学习自己写一个静态页面，最好是有目的性的学习，一开始学html + css的时候就给自己一个目标：我要写一个怎么样的网页出来。比如我要写一个展示我个人信息的静态网站，然后尽量做的漂亮一些。于是在学习的过程中你就可以一点点完善，一点点学得更好。

要学习上面这三个，你可以从写一个微信小程序开始学，有目标，有动力。

如果你要写的不是微信小程序，而是一个正儿八经的网站：


- React
- Vue

这两个都是目前最大最火的前端框架，选一个学就行，对于新手而言个人觉得选哪个都行，不用太纠结。

React 和 Vue要学习的话直接搜索官方文档即可，进官方文档的quickstart跟着一步步学，当然，你也可以看B站。

## 后端语言

- C
一般学校来教入门语言都是这个，好好跟着学校学就行。关键点是指针链表学的扎实一点。然后可以去学C++了。

- C++
重要的面向对象的语言，可以看C++ Primier（协会有这本书）。

- Java
重点是Spring Boot架构

- python
AI常用的语言，爬虫常用的语言，写脚本常用的语言

- Go
很强的语言，在分布式方面应用很广

- lua
胶水型语言，多用于写插件之类的
